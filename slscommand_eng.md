# Additional message structure of the SLS Zigbee Gateway

The data are presented in the format ** topic {payload} **


## Reboot
```
ZigBeeXXXX / reboot {}
```

## Gateway backlight management
```
ZigBeeXXXX / led {“mode”: ”manual”, ”hex”: ”# FFFFFF”}
```
